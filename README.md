## 🛰️ Core Features

### ⚡ The "Hold for Speed" Warp Mechanic
The defining feature of this lab. By holding down on the screen (mouse or touch), the user accelerates their journey through a dynamic 3D starfield.
- **Visual Evolution:** Reaching speeds up to 20x triggers warp trails, intense glitch effects, and hidden messages.
- **Narrative Stakes:** At critical velocities (10x+), the system issues warnings ("SLOW DOWN YOU CRAZY") as the character transitions from stable dancing to a "Stunned" warp state.

### 🎭 Reactive 3D Character (The "Guardian Nova")
Powered by Google's `<model-viewer>`, the character is fully interactive:
- **State-Driven Animation:** Seamlessly transitions between `Initial Hi`, `Idle Dancing`, `Stunned`, and `Recovery` based on user velocity.
- **Parallax Physics:** The character's gaze and rotation react to mouse positions, creating a tactile holographic presence.
- **Creation Origin:** The character was generated using **Rodin AI**, rigged with **Adobe Mixamo**, and converted to GLB format using **Adobe Converter**.

### 📼 Retro TV Aesthetics & Glitch Logic
- **80s Static Entry:** A high-distortion, low-res noise system provides an authentic "TV boot" transition.
- **System Sync:** Includes audio-synced glitch distortions for a "Transmission Received" feel.

### 📜 Holographic Transmission (Bio)
A Star-Lord inspired bio panel detailing:
- **The Tech Arsenal:** MERN Stack, PHP, Laravel, SQL, Java.
- **AI Enhancements:** Python, Hugging Face, Whisper TTS.

## 🛠️ Tech Stack & Engineering

- **Frontend Core:** Vanilla JavaScript (ES6+), HTML5 Canvas (Starfield & Noise).
- **3D Engine:** Google `<model-viewer>` for GLB rendering and physics-aligned rotations.
- **Logic:** High-performance loop handling speed delta-syncing, animation layer switching, and reactive audio playback.
- **Design:** CSS3 Glassmorphism, CSS Glitch Animations, and custom SVG paths for curved text rendering.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Recommended: Latest LTS)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/[YOUR-USERNAME]/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To launch the interactive lab in development mode:
```bash
npm run dev
```

### Production Build
To generate a production-ready optimized version:
```bash
npm run build
```
